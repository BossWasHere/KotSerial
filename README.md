# Serial terminal example
#### Simple serial terminal written in Kotlin

---

## Build only

> ```bash
> $ ./gradlew build

or for Windows

> ```bash
> > gradlew build


## Run
> ```bash
> $ ./run_unix.sh

or for Windows

> ```bash
> > run_windows

Alternatively, run via the JAR file directly after building:

> ```bash
> $ java -jar build/libs/KotSerial-1.0-SNAPSHOT.jar
